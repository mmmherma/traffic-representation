# traffic-representation
Simple polyline representation using HTML, Node.js, Leaflet and Open Street Maps tiles.

# Dependencies
  - [Node.js v10.16.0](https://nodejs.org/en/)
  - [NPM 6.9.0](https://www.npmjs.com/)
  - [Leaflet](https://leafletjs.com/)
